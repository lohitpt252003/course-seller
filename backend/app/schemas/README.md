# Pydantic Schemas

These schemas ensure data validation for input/output in the API.

- **`UserOut`**, **`UserCreate`**, **`UserUpdate`**, **`UserLogin`**: User data handling.
- **`CourseCreate`**, **`CourseOut`**, **`CourseUpdate`**: Course operations.
- **`LessonCreate`**, **`LessonOut`**, **`LessonUpdate`**: Lesson management.
- **`EnrollmentCreate`**, **`EnrollmentOut`**: Enrollment request/response.
- **`PaymentCreate`**, **`PaymentOut`**: Process dummy payments.
- **`ReviewCreate`**, **`ReviewOut`**: Review submission and display.
- **`CertificateOut`**: Certificate data.
- **`CategoryCreate`**, **`CategoryOut`**: Category management.
- **`KeyStats`**, **`AdminStats`**: Admin analytics structures.
