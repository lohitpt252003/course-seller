# Application Pages

Components representing full pages in the React application.

- **`Home/`**: Landing page with hero section and featured courses.
- **`Login/`**: User login form.
- **`Register/`**: User registration form.
- **`Courses/`**: Public course catalog with search and filtering.
- **`CourseDetail/`**: Individual course information, curriculum, and reviews.
- **`Checkout/`**: Dummy checkout flow for enrolling in courses.
- **`CoursePlayer/`**: Learning interface for viewing course content (video/text/PDF).
- **`StudentDashboard/`**: Dashboard for students to view enrollments and progress.
- **`TeacherDashboard/`**: Dashboard for teachers to manage courses and view analytics.
- **`AdminDashboard/`**: Administrative panel for platform management.
- **`Profile/`**: User profile settings.
- **`NotFound/`**: 404 error page.
